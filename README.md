food Gift kitchen
It is a frontend food website designed to showcase and sell beautifully curated gifts for all occasions. It offers an elegant and responsive user interface with categorized gift sections, featured dishes, diner testimonials, and founder details.

Key Features:

Image slider with automatic and manual controls

Gift cuisines for both men and women

Best sellers carousel using Flickity

Stylish diner review cards with social media links

Founder section with bio and links

Responsive design for mobile devices

Built With:
HTML, CSS, and JavaScript

Flickity for carousel effects

Font Awesome for icons

Purpose:
Created as a passion project to merge the love for gifting and design, this website aims to provide a delightful browsing experience for users looking for thoughtful presents.
